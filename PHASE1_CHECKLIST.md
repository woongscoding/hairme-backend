## 💡 팁

### 시간 절약 방법
1. **백엔드 먼저 완료**: 안드로이드 개발 전에 API가 준비되어야 함
2. **단계별 테스트**: 각 단계마다 바로 테스트
3. **간단한 UI**: 복잡한 애니메이션은 나중에
4. **개인정보처리방침**: 템플릿 활용

### ML 엔지니어 관점에서 배울 점
1. **데이터 파이프라인**: 실제 사용자 데이터 수집 경험
2. **AB 테스트**: OpenCV vs Gemini 비교
3. **프로덕션 모니터링**: CloudWatch 로그 분석
4. **피드백 루프**: 사용자 피드백 → 모델 개선
